# Vcdp是什么？




我们很高兴为您介绍我们的所见即所得视频编辑器。该编辑器采用Typescript编写，使用rollup进行打包，并集成了Konva图形库以实现强大的绘图功能。

Vcdp（video composition demo player）是一个现代的、使用Typescript编写、基于[Konva](https://konvajs.org/)实现的所见即所得的视频编辑器。

Vcdp使用Typescript编写，所以它具有框架无关性，您可以随意的将它放在您的ReactJS、VueJS、solidJS项目中，或者您没有使用任何现代的Javascript框架

该编辑器的主要目标是使视频编辑变得简单而直观。它提供了一个易于使用的界面，让您轻松地添加和调整视频、图像、文本和音频元素，从而创建出令人惊叹的视频内容。

通过使用Konva图形库，我们能够提供出色的绘图性能和灵活性，让您能够自由地创造出各种形状和效果。同时，该编辑器还支持各种常见的视频编辑功能，如剪切、复制、粘贴和调整时间轴等。

最重要的是，该编辑器是开源的，我们欢迎开发者和爱好者的加入和贡献。我们相信这是一个有前途的项目，它将在未来的视频编辑领域发挥着重要的作用。
